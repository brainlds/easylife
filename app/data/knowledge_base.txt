# 密码重置
如何重置密码？
1. 打开登录页面
2. 点击"忘记密码"链接
3. 输入您的注册邮箱
4. 点击"发送重置链接"
5. 在邮箱中查收重置邮件
6. 点击邮件中的链接
7. 设置新密码

# 系统优化
系统性能优化指南：
1. 清理磁盘空间
   - 删除临时文件
   - 卸载不用的程序
   - 清理下载文件夹

2. 优化启动项
   - 检查开机启动项
   - 关闭不必要的后台程序
   - 更新系统驱动

3. 硬件升级建议
   - 增加内存容量
   - 更换SSD硬盘
   - 升级处理器散热

# 账户安全
账户安全建议：
1. 使用强密码
   - 至少8位字符
   - 包含大小写字母
   - 包含数字和特殊符号

2. 开启两步验证
   - 登录时需要验证码
   - 支持手机短信验证
   - 支持认证器APP

3. 定期更换密码
   - 建议每3个月更换
   - 不要使用重复密码
   - 保存好密码记录

# 常见问题
Q: 登录时提示"账户已锁定"怎么办？
A: 账户连续输错密码5次会被锁定，请等待30分钟后再试，或联系客服解锁。

Q: 如何修改绑定手机号？
A: 请登录账户中心，在"安全设置"中找到"手机绑定"选项进行修改。

Q: 无法收到验证码怎么办？
A: 请检查：
1. 手机是否有信号
2. 是否被拦截为垃圾短信
3. 手机号是否正确
如果仍有问题，请联系客服。
